# Neovate Settings 配置查询

当用户询问 Neovate Code 的配置、设置、settings 相关问题时使用此技能。

## 触发场景

- 询问某个配置项的含义（如"approvalMode 是什么"）
- 询问如何配置某功能（如"怎么设置代理"、"怎么配置 MCP"）
- 询问配置的默认值、类型、可选值
- 询问 Neovate 设置相关的任何问题

## 配置索引

| 配置项 | 用途 |
|--------|------|
| approvalMode | 审批模式控制 |
| agent | 子代理配置 |
| autoCompact | 自动紧凑历史记录 |
| autoUpdate | 自动更新开关 |
| browser | 浏览器 MCP 集成 |
| commit | 提交配置 |
| desktop | 桌面应用设置 |
| extensions | 第三方扩展配置 |
| httpProxy | HTTP 代理设置 |
| language | 界面语言 |
| mcpServers | MCP 服务器配置 |
| model | 主模型设置 |
| notification | 通知设置 |
| outputFormat | CLI 输出格式 |
| outputStyle | 输出样式 |
| planModel | 规划模型 |
| plugins | 插件列表 |
| provider | 提供商配置 |
| quiet | 静默模式 |
| skills | 技能文件路径 |
| smallModel | 轻量模型 |
| systemPrompt | 自定义系统提示 |
| temperature | 温度参数 |
| todo | 待办事项功能 |
| tools | 工具开关 |
| visionModel | 视觉模型 |

详细配置请查阅 configs/ 目录下对应文件。
